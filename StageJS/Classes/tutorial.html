    The main of this website is to discover and learn the JavaScript language. For that, we have created some classes to simplify the manipulation of different objects like a pixel, a picture, a sound or also a video.

    What is a pixel?

    A pixel is defined by a three integer values: red, green and blue between 0 and 255. For example, white pixel was defined by (255, 255, 255), black pixel by (0, 0, 0), classic red pixel by (255, 0, 0), classic green pixel by (0, 255, 0) and classic blue pixel by (0, 0, 255).

    Another value, less known and used is the integer of alpha also between 0 and 255. It’s the percentage of transparency of the pixel. The value 0 equals a transparency of 100%, so a transparent pixel. The value 255 is a pixel totally visible, so a transparency of 0%. The alpha can be used for example to modify the light of the pixel.

    A pixel alone can’t be call a picture, so a picture is composed of many pixel. To know where the pixel which we would modify is, it’s important to know its position in the width (x) and height (y) of the picture.

    Example of a red pixel 50% transparent: (255, 0, 0, 127) Example of a black pixel 50% transparent: (0, 0, 0, 127)

    What is a picture?

    A picture is defined with its width and height values both as integer. There are the number of pixel in the width and in the height in the pixel. In fact, a picture is an array of pixel with width*height pixel in total. A picture is write and read to the top corner left to the bottom corner right, as a book.     Property and functions of the class Pixel in our JavaScript

    As said, a pixel need some values: red, green, blue, alpha and its position x and y in the picture. The constructor of the class Pixel need the same number and order of properties. Of course, it isn’t possible to take a values smaller than 0 and higher than 255 for a colour. It is possible to reach all of them, with the call of property (“this.”) or with a function. To create a pixel black at coordinates 50, 50: var myPixel = Pixel(0, 0, 0, 255, 50, 50); To display some information about a pixel: myPixel.toString(); To reach the value red with property: myPixel.this.red; To reach the value green with property: myPixel.this.green; To reach the value blue with property: myPixel.this.blue; To reach the value alpha with property: myPixel.this.alpha; To reach the position x with property: myPixel.this.x; To reach the position y with property: myPixel.this.y; To reach the colour with function: myPixel.getColor(); To reach the position with function: myPixel.getPos(); All of this properties can be modify, just with the call of this property or with a function which we have created. To modify the red value of the pixel with property: myPixel.this.red = 255; To modify the alpha value of the pixel with property: myPixel.this.alpha = 175; To modify all colours of the pixel to white with function: myPixel.setColor(0, 0, 0, 255); To modify all coordinates of the pixel with function: myPixel.setPos(100, 100);

    Property and functions of the class Picture in our JavaScript

    Before to draw or modify a picture, the picture must be existed. Create a picture needs an image in the same folder source as parameter. The format of the image can be a “png” or “jpg”. The path source of the picture can be modify. To use the picture, she is drawing in a canvas, a space in the web page. The size of this canvas was initialised to the size of the picture, but it is possible to modify it. To create a picture: var myPicture = Picture(“picture.jpg”); To display the first time a picture: myPicture.display(); To draw the others time the picture: myPicture.redraw(); To display some information about the picture: myPicture.toString(); To reach the width with property: myPicture.this.width; To reach the height with property: myPicture.this.height; To reach the canvas with property: myPicture.this.canvas; To reach the path source with property: myPicture.this.source; To reach the colour of a pixel with function: myPicture.getPixel(50, 50);
